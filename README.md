<div>
  <h1 align="center">Features</h1>
  <p>
    <h2>Downloads</h2>
    <p>
      Downloads for these applications are provided, all with searches for the latest versions (besides moreclocktool for compatibility)
      <img width="1202" height="1018" alt="image" src="https://github.com/user-attachments/assets/e1ffbc88-96fe-4434-bf2e-687f2612055c" />
    </p>
    <h2>Tweaks</h2>
    <p>
      Buttons are provided to apply / undo tweaks from OSlivion
      <img width="1202" height="682" alt="image" src="https://github.com/user-attachments/assets/b6b52565-ab5e-4c29-847e-7089e25b26b6" />
    </p>
    <h2>Tools</h2>
    <p>
      As of now, one tool is provided:
      Auto Timer Resolution: Automatically starts a stress tests and tests a given range of timer resolutions and applies the best one for you.
    </p>
  </p>
  <br>
  <p>
    <h1>Setup</h1>
    Extract the OSO.zip and put it (preferably) in C:\Oslivion\OSO\OSO.exe for compatibility, it should work everywhere else but thats the working directory it is built for.
  </p>
